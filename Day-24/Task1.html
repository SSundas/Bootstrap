Slider/Carousel

Three main parts;

  1. Images
  2. Controls
  3. Indicators

#myCarousel

Helps to function and controls properly.

class="carousel"

It specifies that this div cintains carousel.

class="slide"

Slide class in carousel is use to perform animations.

Data-ride

This attribute tells Bootstrap to begin animating the carousel immediately when the page loads..

Indicators Part

The indicators are the little dots at the bottom of each slide.Which indicates how many slides there are in the carousel,and which slide the user is currently viewing.

  • Carousel Indicators
  • Data target
  • Data-Slide-To

*Carousel Indicators

The indicators are specified in an ordered list with class.

*Data target

The data-target attribute points to the id of the carousel.

*Data-Slide-To

The data-slide-to attribute specifies which slide to go to, when clicking on the specific dot.

Wrappers For Slides

  • Carousel Inner
  • Item
  • Active

*Carousel inner

The slides are specified in a div with class.

*Item

The content of each slide is defined in a div with class.

*Active

This class needs to be added to one of the slides. Otherwise, the carousel will not be visible.

Controls

"Left and Right controls"

This code adds "left" and "right" buttons that allows the user to go back and forth between the slides manually. The data-slide attribute accepts the keywords "prev" or "next". Which alters the slide position relative to its current position.
